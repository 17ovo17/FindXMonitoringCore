## 2. 运行态门禁

- 用户、团队、角色、站点设置、变量设置、单点登录、告警引擎均可从参考站访问。
- AI 配置在参考站不可作为运行态完成证据，必须以本地源码和 FindX 自有系统配置实现为事实源。
- 所有页面必须保留成熟页面结构、筛选、表格、树、详情、表单、弹窗、权限态、空态和错误态。
- FindX 用户侧不能出现外部品牌菜单、页面标题、权限对象或审计对象。
