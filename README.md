# FindX

FindX 是面向企业运维和可观测场景的一体化平台，覆盖基础监控、日志中心、链路监控、CMDB 资产、Agent 管理中心、AI SRE、巡检诊断和 Evidence Chain。

平台目标是把指标、日志、Trace、资产、告警、Agent 状态、配置变更和诊断证据统一到 FindX 控制面中，形成可登录、可导航、可审计、可回溯、可持续演进的完整运维平台。
