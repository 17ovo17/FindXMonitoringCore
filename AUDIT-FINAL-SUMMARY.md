## 核心设计决策（已确认）

1. **主题自己的，功能像素级对齐源码**
2. **Context Engineering 贯穿全局**（不是简单 CoT）
3. **工作流不暴露前端**（后端自动调用）
4. **AI 对话放导航第一位**
5. **Catpaw 纯执行层，LLM 统一调度**
6. **凭证跟着 CMDB 资产走**（有就自动填充，没有提示配置）
7. **Catpaw 默认启用核心插件，按 CMDB 类型推荐其他**
8. **CMDB 单机 AI 排障 = 主页 AI 同一个 LLM，只是注入不同上下文**
9. **沙箱三级安全分层**（Level 0 只读自动 / Level 1 低风险通知 / Level 2 确认 / Level 3 禁止）
10. **夜莺 API + MCP 直接对接，不重写已有能力**
11. **多平台 Gateway**（飞书/企微/钉钉，参考 Hermes 源码）
12. **AI 必须能调用日志和链路**（可观测三支柱完整覆盖）
13. **拨测功能已在架构中**（基于 n9e Probe 体系）
14. **知识库独立一级导航**（6 个子库 + 自我学习闭环）
